# geopolitics - 地政学と半導体産業

## 概要

このセクションでは、**半導体産業における地政学的影響**を扱います。  
米中対立、台湾有事リスク、EUV輸出規制、Rapidus構想など、  
**国家戦略・外交・安全保障と技術の接点**に注目し、技術者の視点から複眼的に解釈します。
