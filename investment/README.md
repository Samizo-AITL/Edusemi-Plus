## 🧭 概要 | Overview

本ディレクトリは、半導体技術に関連する**産業構造・市場動向・投資戦略**を体系的にまとめた教材群です。  
Edusemiの技術的枠組みに対して、投資・経済・地政学の視点から拡張された「Edusemi-Plus」構成の一部です。

---

## 🎯 目的 | Purpose

- 技術と株式市場の連動を理解する  
- 半導体企業のビジネスモデルと投資評価を整理する  
- ETF・個別株・マクロ要因を踏まえた実践的投資戦略を検討する
