# 🗂️ Edusemi-Plus Archive

> 本ディレクトリは、**Edusemi-Plus教材**の補完資料として、  
> 半導体技術・設計思想・産業史の文脈で重要な**技術記録・エピソード・ドキュメント**を保存するアーカイブ領域です。

---

## 📘 目的

- **技術史的価値を持つ資料の保管**
- 教材本編に収まりきらない**体験知・設計思考・工程知識**の記録
- ChatGPT等を用いた**教材生成・プロンプト設計・演習化**の素材とする
