## 📑 説明 / Description
コネクタは**嵌合・接触・保持力**などの機構要件と、**差動ペア配列・リターンパス・接触抵抗**といった電気要件を同時に満たす必要があります。  
*They must satisfy mechanical needs (mating, retention) and electrical needs (differential pinout, return paths, low contact resistance).*
