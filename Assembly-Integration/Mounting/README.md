## 🏗 概要 / Overview
実装方式は、半導体チップや受動部品を基板上に固定・接続するための技術です。  
*Mounting technologies define how semiconductor chips and passive components are fixed and connected on a PCB.*  

代表的な方式として、表面実装技術 (SMT)、チップサイズパッケージ (CSP)、ボールグリッドアレイ (BGA) などがあり、信頼性・熱特性・歩留まりに大きな影響を与えます。  
*Typical methods include Surface Mount Technology (SMT), Chip Size Package (CSP), and Ball Grid Array (BGA), each strongly influencing reliability, thermal performance, and yield.*
