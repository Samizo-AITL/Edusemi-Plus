# 🤖 ai-semiconductor – AI時代の半導体を読む

本シリーズは、**生成AI・大規模言語モデル（LLM）・NPU・GPU** といったトピックを通して、  
**AIと半導体の交点を多面的に読み解く教育・分析モジュール**です。

焦点は「技術」そのものではなく、  
**技術がいかに産業・製品・戦略と結びつき、世界を変えているか**にあります。  

設計者・経営層・戦略担当者・研究者など、  
**半導体ビジネスを深く考えるすべての人**に向けて構成されています。
